# 🔬 مساعد الباحث - Research Agent Interface

واجهة تفاعلية سهلة الاستخدام لتحليل ومناقشة المستندات البحثية باللغة العربية.

## 🎯 المميزات

### 💬 مناقشة تفاعلية
- طرح أسئلة مباشرة حول محتوى المستند
- إجابات ذكية باللغة العربية
- محفوظات المحادثة

### 📊 تحليل شامل
- تحليل تلقائي للمحتوى
- تقييم مستوى الأهمية العلمية
- تحليل إحصائي متقدم (SciPy)
- رسوم بيانية تفاعلية

### 🔍 استكشاف المحتوى
- بحث ذكي في النص
- إحصائيات الكلمات
- تجزئة المحتوى

### 📄 تقارير وتصدير
- تقارير مفصلة بصيغة Markdown
- تصدير البيانات كـ JSON
- إحصائيات شاملة

## 🚀 كيفية التشغيل

### الطريقة السريعة
```bash
# شغل الملف التنفيذي
start_interface.bat
```

### التشغيل اليدوي
```bash
# فعّل البيئة الافتراضية
.venv\Scripts\activate

# تثبيت المتطلبات
pip install streamlit plotly

# تشغيل الواجهة
streamlit run src/app_streamlit.py
```

## 📱 كيفية الاستخدام

### 1. رفع المستند
- اضغط على "اختر ملف PDF" في الشريط الجانبي
- حدد ملف PDF من جهازك
- اضغط "تحليل المستند"

### 2. المناقشة التفاعلية
- انتقل إلى تبويب "💬 المناقشة"
- اكتب سؤالك في مربع النص
- احصل على إجابة فورية من المساعد الذكي

### 3. استكشاف التحليل
- تبويب "📊 التحليل": شاهد النتائج والرسوم البيانية
- تبويب "🔍 الاستكشاف": ابحث في المحتوى
- تبويب "📄 التقارير": صدّر النتائج

## 🛠️ المتطلبات التقنية

- Python 3.8+
- Windows 10/11
- متصفح ويب حديث
- اتصال إنترنت (لـ OpenRouter API)
